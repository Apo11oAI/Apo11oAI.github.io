# 诊断问题
> 病人来了肯定是病了。
>
> 问题是:
>
> 患者得的是什么病?
> 到那个阶段了?
> 有没有并发症?
> 有没有可能误诊\漏诊?

1. 在患者可接收范围内搜集有意义的信息
    1. 初始信息的搜集
        - 症状
        - 体征 
    2. 补充检查手段的选择
    3. 病史整理
2. 推断阶段
    1. 病理生理机制的形成
    2. 鉴别诊断
