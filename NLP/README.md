# 自然语言
1. 是传递思维的表示方法、约定、规则的集合
2. 由语句组成、语句由词组成
3. 遵循一定的语法、语义规则
4. 词汇和语法体系是构成自然语言的两大要素。

# 自然语言处理
## 主要任务
1. 输入自然语言（听读）
    - 语言表示（数据结构）
        - 字符串（字符数组）
2. 理解自然语言
    - 语言是思维的载体，理解语言就是理解思维。
3. 输出自然语言（说写）
    - 翻译、释义
    - 回答
    - 摘要
